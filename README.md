# Vibe Commerce – Mock E-Com Cart

Full-stack shopping cart app 

## Tech Stack
- **Frontend**: React
- **Backend**: Node.js + Express
- **Database**: SQLite (file-based, no setup needed)
- **APIs**: RESTful

## Features
- Browse 5 mock products
- Add/remove items from cart
- View cart total
- Checkout with name/email → mock receipt
- Responsive design
